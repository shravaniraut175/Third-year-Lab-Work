🧾 EXPERIMENT 1


This code loads the Iris dataset, handles missing values, normalizes the data, and visualizes it using line and histogram plots.


import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from sklearn.datasets import load_iris


# Load dataset
data = load_iris()
df = pd.DataFrame(data.data, columns=data.feature_names)


# Add missing values artificially
df.iloc[0,0] = np.nan


# Handle missing values
df.fillna(df.mean(), inplace=True)


# Normalize data
df_norm = (df - df.min())/(df.max()-df.min())


# Graphs
plt.plot(df['sepal length (cm)'])
plt.title("Sepal Length Plot")
plt.show()


plt.hist(df['sepal width (cm)'])
plt.title("Histogram")
plt.show()




🧾 EXPERIMENT 2


Linear regression is used to find the relationship between two variables and plot the best-fit line.


from sklearn.datasets import load_iris
from sklearn.linear_model import LinearRegression
from sklearn.model_selection import train_test_split
import matplotlib.pyplot as plt


data = load_iris()
X = data.data[:,0].reshape(-1,1)
y = data.data[:,1]


X_train, X_test, y_train, y_test = train_test_split(X,y,test_size=0.2)


model = LinearRegression()
model.fit(X_train,y_train)


y_pred = model.predict(X_test)


plt.scatter(X_test,y_test,color='red')
plt.plot(X_test,y_pred,color='blue')
plt.title("Linear Regression")
plt.show()


🧾 Exp 3
Logistic regression classifies data into categories using a sigmoid function.


from sklearn.datasets import load_iris
from sklearn.linear_model import LogisticRegression
from sklearn.model_selection import train_test_split


data = load_iris()
X = data.data
y = data.target


X_train,X_test,y_train,y_test = train_test_split(X,y,test_size=0.2)


model = LogisticRegression(max_iter=200)
model.fit(X_train,y_train)


print("Accuracy:",model.score(X_test,y_test))




🧾 exp 4 - svm
SVM finds the best boundary (hyperplane) to separate different classes.
from sklearn.datasets import load_iris
from sklearn.svm import SVC
from sklearn.decomposition import PCA
import matplotlib.pyplot as plt


data = load_iris()
X = data.data
y = data.target


# Reduce to 2D for graph
pca = PCA(n_components=2)
X_pca = pca.fit_transform(X)


model = SVC(kernel='linear')
model.fit(X_pca,y)


plt.scatter(X_pca[:,0],X_pca[:,1],c=y)
plt.title("SVM Decision Boundary")
plt.show()

Hebbian learning updates weights based on input-output correlation (“fire together, wire together”).


And gate
import numpy as np
import matplotlib.pyplot as plt


X = np.array([[0,0],[0,1],[1,0],[1,1]])
y = np.array([0,0,0,1])
w = np.zeros(2)


for i in range(len(X)):
    w += X[i]*y[i]\

🧾 EXp 6 - EM
EM algorithm groups data into clusters by iteratively estimating and updating parameters.
import numpy as np
import matplotlib.pyplot as plt
from sklearn.mixture import GaussianMixture


# Generate data
X = np.concatenate([np.random.normal(2,1,200),
                    np.random.normal(-2,1,200)]).reshape(-1,1)


# Model
model = GaussianMixture(n_components=2)
model.fit(X)


# Plot
plt.hist(X,bins=30)
plt.title("Data Distribution")
plt.show()




🧾 exp 7
MCP model uses threshold logic to simulate neurons and implement logical gates.
def mcp(x1,x2,theta):
    return 1 if (x1+x2)>=theta else 0


inputs = [(0,0),(0,1),(1,0),(1,1)]


print("AND Gate")
for i in inputs:
    print(i,mcp(i[0],i[1],2))


print("OR Gate")
for i in inputs:
    print(i,mcp(i[0],i[1],1))
